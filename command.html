Commands

Following are the basic git commands:

  • To make git repository

    git init

  • To clone a remote repository

    git clone URL-of-remote-repository

  • To add a file

    git add Name-of-file

  • To add multiple files

    git add -A

  • To commit the changes

    git commit -m "Commit Message (Manadatory)"

  • To view status of repository

    git status

  • To check log of commits

    git log
